Taro 脚手架 demo
------------
Taro 脚手架，最新版本支持 Taro 3.0 
支持数据mock 方便自己写代码调试

整合
------------
* dev环境支持 数据mock
* 支持 DVA 
* 支持 redux
* 支持 template 脚手架
	* node template.js 模块名称
	```
	node template.js test
	```


#### 微信小程序
* `git clone https://github.com/firma/trao-demo.git`
* `npm install || yarn install`
* `yarn dev:weapp` 启动服务
*  用微信开发者工具打开dist目录

#### 头条小程序
* `yarn dev:tt`
* 头条小程序开发工具打开dist目录

#### h5
* `yarn dev:h5` 启动服务
* 打开浏览器
